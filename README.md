# Number Guessing Game

A simple number guessing game in Python with both CLI and GUI versions.

## Features
- Command-line interface version
- GUI version with tkinter
- Multiple difficulty levels
- Attempt tracking

## How to Play

### CLI Version
```bash
python game.py
```

### GUI Version
```bash
python game_gui.py
```

## Requirements
- Python 3.x
- tkinter (included with Python)
